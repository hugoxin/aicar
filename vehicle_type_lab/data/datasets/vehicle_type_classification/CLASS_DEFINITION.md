# Class Definition

This document defines the first three vehicle classes for wash strategy selection.

## sedan

普通轿车，车身较低，三厢/两厢轿车均可先归入 `sedan`。

## suv

SUV、城市越野、跨界 SUV，车身较高。

## mpv

MPV、商务车、多人座厢式车辆，车身更长更高，车顶较平。

## Not Handled Yet

The current three-class scope does not handle:

- `pickup`
- `van`
- `truck`
- `bus`
- `sports car`
- `unknown`

These categories can be expanded in later stages after the first sedan / suv / mpv flow is stable.
