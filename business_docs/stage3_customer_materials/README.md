- 一页式项目介绍
- 客户演示话术
- 演示大纲
- 常见问题
- 价值主张
- 技术边界说明
- 后续路线图

这些材料基于当前三个冻结基线：

- `stage1-frozen-baseline`
- `stage2-simulation-baseline`
- `stage3-visual-baseline`

配套展示页位于：

