# Viewer 能力边界

## Viewer 会做

- 读取冻结 Stage4.5-R 路径结果。
- 导出稳定、简单的网页数据合同。
- 显示通用 MPV、状态路径、扫描点、尾迹和播放信息。
- 对路径与车体应用一致的展示缩放。

## Viewer 不会做

- 不改变、优化或重新运行路径算法。
- 不对展示缩放后的 MPV 重新做机械或碰撞安全验证。
- 不把展示缩放解释为真实机械轨迹。
- 不代表任何品牌或真实 MPV 的 CAD、尺寸标定与喷头可达性。
- 不连接 PLC、伺服、机器人、传感器或其他硬件。
- 不生成可下发的速度、位置或控制指令。
- 不用于真实设备决策。

Stage4.5 的安全指标不能直接套用到经过 display transform 的 MPV 展示几何。Stage4.6 数据接口只预留，不属于 Viewer V1 正式输入。
