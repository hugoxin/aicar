# System Architecture

AI 智能无人洗车系统可以分为四层。

## App / 后台

未来业务层，用于订单、用户、设备状态、运营看板和远程配置。

## Python 上位机

算法和流程层，用于车辆识别、车辆模型选择、路径规划、洗车流程状态机、日志和仿真。

## VirtualPLC / PLC

执行和安全层，用于互锁、急停、泵阀控制、限位、传感器状态和异常保护。

## 机械 / 水泵 / 阀门 / 传感器

设备层，用于真实执行清洗动作和采集现场状态。

## Current Scope

当前阶段只做 Python 仿真和 AI 小闭环的目录框架。暂不连接真实硬件，不控制水泵、阀门、传感器或 PLC。
