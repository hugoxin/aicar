# Roadmap

## Phase 0: 目录与框架搭建

创建总工作区、子项目边界、配置、文档、占位代码和检查脚本。

## Phase 1: 车辆识别三分类小 demo

在 `vehicle_type_lab` 中做 sedan / SUV / MPV 三分类小闭环。先从少量样本和清晰输出格式开始，不追求复杂模型。

## Phase 2: 无人洗车 2D/2.5D 纯仿真

在 `aicar_sim` 中建立车辆尺寸、洗车房坐标、喷嘴安全距离、基础路径和状态机仿真。

## Phase 3: AI 识别结果接入仿真

让 `vehicle_type_lab` 输出 JSON 结果，`aicar_sim` 读取结果并选择车辆模型和路径策略。

## Phase 4: VirtualPLC

建立 VirtualPLC 接口和安全状态模拟，用于学习 PLC 逻辑、互锁和异常处理。

## Phase 5: 小比例实验台

在不接触真实车辆的前提下，用小比例机构验证流程、传感器和执行器控制逻辑。

## Phase 6: 真实车身局部实验

只针对真实车身局部区域做低风险测试，重点验证安全距离、喷嘴覆盖和异常停机。

## Phase 7: 商业化原型

在技术、成本、安全、合作和知识产权边界明确后，再推进商业化原型。
