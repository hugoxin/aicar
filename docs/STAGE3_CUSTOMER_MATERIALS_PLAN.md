# Stage3 Customer Materials Plan

## 1. 阶段3.4定位

阶段3.4是客户演示材料整理阶段。它不是新算法，不是新 Demo，不是 PPTX 制作，不修改核心代码，也不进入硬件控制。

本阶段只整理文档、话术、演示流程、问答材料、价值主张和技术边界说明。

## 2. 为什么需要客户演示材料

阶段1、阶段2、阶段3和阶段3.3已经让项目具备可运行、可展示的软件闭环。客户演示材料的目标，是把工程成果转换成更适合对外沟通的语言。

它帮助回答：

- 项目到底做了什么？
- 当前已经做到哪里？
- 当前 Demo 能证明什么？
- 当前还不能证明什么？
- 后续继续投入可以走向哪里？

## 3. 与 stage3_customer_showcase_demo 的关系

`demos\stage3_customer_showcase_demo` 是可运行的一页式 HTML 展示页。

`business_docs\stage3_customer_materials` 是配套话术和材料包，服务于现场讲解、客户问答、PPT大纲和商业价值表达。

两者关系是：

```text
Stage3.3 HTML 展示页 = 可看的页面
Stage3.4 客户材料包 = 可讲的材料
```

## 5. 当前适合用于什么场景

- 客户初次沟通
- 项目组内部汇报
- 领导评审
- 合作方技术交流
- 项目申报前期材料准备
- 后续 PPT/PDF/视频脚本的素材基础

## 6. 当前不能替代什么

- 不能替代正式商业计划书
- 不能替代 PPTX 成稿
- 不能替代真实硬件测试报告
- 不能替代 PLC 联调方案
- 不能替代商业级 AI 模型评估报告
