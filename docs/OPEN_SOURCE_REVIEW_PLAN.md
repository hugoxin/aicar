# Open Source Review Plan

当前只做目录和计划，不下载、不复制代码。

后续准备参考：

- `Turntable_carwash`：洗车设备结构、电气、PLC 参考。
- `PythonRobotics`：路径规划、覆盖路径参考。
- `pymodbus`：Python 与 PLC 通讯参考。
- `openplc-runtime`：虚拟 PLC 学习参考。
- `python_motion_planning`：路径规划学习参考，但需要特别注意 license。

## Review Rules

- 第三方源码只放在 `external_repos`。
- 不把第三方源码直接复制进 `aicar_sim` 或 `vehicle_type_lab`。
- 阅读和借鉴前先确认 license。
- 需要商业化前，再做一次系统的 license 和知识产权审查。
